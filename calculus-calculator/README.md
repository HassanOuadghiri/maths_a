# Calculus Calculator

This project is a web application designed to calculate limits, derivatives, and integrals, providing users with visualizations and step-by-step solutions. 

## Features

- **Limit Calculation**: Evaluate limits at specific points with detailed steps.
- **Derivative Calculation**: Compute derivatives of functions using various differentiation rules.
- **Integral Calculation**: Perform definite and indefinite integrals with clear explanations.
- **Graph Visualization**: Visualize functions, their derivatives, and integrals using interactive graphs.
- **User-Friendly Interface**: Intuitive input fields and organized display of results.

## Technologies Used

- React: For building the user interface.
- Charting Library: For rendering graphs of functions.
- CSS: For styling the application.

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd calculus-calculator
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```
Open your browser and go to `http://localhost:3000` to view the application.

## Usage

1. Enter a mathematical function in the input field.
2. Select the operation (limit, derivative, or integral).
3. Click the "Calculate" button to see the results.
4. View the graphical representation of the function and its calculations.
